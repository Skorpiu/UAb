## Conteúdo

### mdc.py
Implementação do algoritmo de Euclides para cálculo do Máximo Divisor Comum (MDC) entre dois números inteiros.

## Como Executar

```bash
python src/mdc.py <número1> <número2>
```

## Exemplo

```bash
python src/mdc.py 48 18
# Saída: MDC(48, 18) = 6
```
