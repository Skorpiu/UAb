# Fundamentos de Gestão (C#)

Repositório contendo os trabalhos práticos da unidade curricular de Fundamentos de Gestão, desenvolvidos em C#.

## Estrutura do Projeto

```
uab-2-fg-cs/
├── src/           # Códigos-fonte dos projetos
├── docs/          # Documentação
└── relatorios/    # Relatórios dos trabalhos
```

## Conteúdo

Este repositório contém implementações de conceitos fundamentais de gestão, incluindo:
- Gestão de recursos
- Análise financeira
- Tomada de decisão
- Planejamento estratégico

## Requisitos

- .NET Core 3.1 ou superior
- Visual Studio 2019/2022 ou Visual Studio Code
- SQL Server (para projetos com banco de dados)

## Como Executar

1. Navegue até a pasta do projeto:
   ```
   cd src/NomeDoProjeto
   ```

2. Restaure os pacotes NuGet:
   ```
   dotnet restore
   ```

3. Execute o projeto:
   ```
   dotnet run
   ```
